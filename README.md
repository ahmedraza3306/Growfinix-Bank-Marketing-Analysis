# Bank Marketing Analysis

## Project Overview

This project analyzes the Bank Marketing Dataset from the UCI Machine Learning Repository. The dataset contains information about bank customers and whether they subscribed to a term deposit.

The analysis focuses on customer demographics, education levels, job distribution, age distribution, and deposit subscription trends.

## Dataset

Source: UCI Machine Learning Repository

Dataset Size:

* 45,211 Records
* 17 Features

## Technologies Used

* Python
* Pandas
* Matplotlib

## Analysis Performed

1. Customer Age Analysis
2. Job Distribution Analysis
3. Education Level Analysis
4. Deposit Subscription Analysis
5. Data Visualization

## Key Insights

* Most customers belong to a few dominant job categories.
* Secondary education is the most common education level.
* The majority of customers did not subscribe to a term deposit.
* Customer age distribution is concentrated around middle-aged groups.
